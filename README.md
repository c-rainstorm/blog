## Tips:

- 为方便查阅博客，可以在浏览器安装 [Octotree](https://github.com/buunguyen/octotree) 插件
- 目录生成方式
  1. blog 根目录执行【`tree -H . . -I 'out|res'`】，得到HTML输出
  2. 将 `body` 里目录部分复制过来
